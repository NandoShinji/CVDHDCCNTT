# API

### How to run Project?

-- install Nodejs: https://nodejs.org/en/
-- install AdonisJS: npm i --save @adonisjs/cli
-- install Package: npm install
-- Run: adonis serve --dev

### API

- Endpoint: /api/v1/sensor
